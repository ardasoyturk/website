## 1. Fast Context

I wanted one file that includes:

- Long paragraphs for rhythm and line length testing
- Dense lists with nested bullets
- Tables, quotes, and callout blocks
- Code fences with different languages
- Details/summary blocks and inline HTML

If this page looks clean, most future posts will look clean.

> Good writing needs good defaults. Great defaults make writing feel effortless.

## 2. Mixed Content Test

### Short paragraph

Modern personal blogs should feel readable at `08:00` on mobile and `23:30` on desktop. Same content, different context.

### Long paragraph

There is a specific kind of calm that comes from a page where every layer is intentional: typography that breathes, contrast that does not fight your eyes, spacing that creates momentum, and components that disappear into the writing instead of shouting over it. This post exists to verify exactly that.

## 4. Table

| Layer      | Goal        | Rule                            |
| ---------- | ----------- | ------------------------------- |
| Typography | Readability | Keep line-length around 65-75ch |
| Color      | Comfort     | Strong contrast, soft surfaces  |
| Motion     | Clarity     | Subtle only, no visual noise    |
| Components | Focus       | Style content, not chrome       |
